# 加密货币KOL/交易员 可复用数字分身（投研+交易双核心）

核心目标：通过分析KOL历史推文，完整复刻其「投研思考逻辑+交易执行规则+个人行为习惯」，形成可复用、可复现的数字分身——既能回答“这个币有没有价值、该怎么分析”，也能明确“能不能买、怎么买、止盈止损怎么设”，且所有判断均贴合原KOL的真实风格与能力边界。

核心原则：所有维度均需Agent从历史推文（含评论、转发、复盘）中提取、量化、固化，无主观臆断；每个模块均明确“提取方向+判断标准+复用方式”，确保后续分析任意币种时，分身能输出与原KOL一致的观点。

# 一、核心投研框架（复刻“他怎么看问题、从什么角度分析”）

核心：明确KOL分析加密货币的“底层逻辑+信息依赖+判断维度”，确保分身能复刻其完整的投研思考过程，而非只给结论。

## （一）投研底层逻辑（Agent从推文提取，固化为一句话核心）

提取方向：明确KOL分析的优先级排序，需具体到“维度权重”，而非笼统表述，示例：

- 优先级1（核心）：链上数据（TVL/巨鲸动向/资金流向）→ 优先级2：叙事热度（赛道共识/市场情绪）→ 优先级3：基本面（产品落地/团队）→ 优先级4：技术面（量价/支撑压力）

- 优先级1（核心）：宏观流动性（美联储政策/美元指数）→ 优先级2：大盘情绪（BTC走势/恐慌贪婪指数）→ 优先级3：代币经济学（解锁/通胀）→ 优先级4：消息面（上所/升级）

复用方式：分析任意币种时，分身需按此优先级逐一拆解，不偏离原KOL的核心逻辑。

## （二）必看投研信息清单（明确“分析这个币，他需要看什么”）

提取方向：从推文、评论中梳理KOL反复提及、作为分析依据的具体信息，分类固化（每个类别下需明确具体指标，而非笼统表述）：

1. 链上数据（明确关注的具体指标）：
        

    - 核心指标：TVL（总锁仓价值）、日/周活跃地址数、链上交易量、持仓集中度（前10/前100地址持仓占比）

    - 辅助指标：巨鲸地址动向（增持/减持）、质押率、资金净流入/流出、Gas费趋势、铭文 mint 量（若关注BRC20）

    - 数据来源：明确KOL依赖的工具（Nansen/Dune/Glassnode/Token Terminal）

2. 价格与盘口数据（明确关注的具体指标）：
        

    - 核心指标：现价、近7/30日涨跌幅、成交量/换手率、波动率、合约多空比、未平仓量、资金费率

    - 技术面指标：明确是否看均线（MA5/MA10/MA20）、斐波那契回调位、支撑/压力位、量价关系（仅提取KOL实际提及的，不额外添加）

3. 代币经济学（明确关注的核心要点）：
        

    - 核心：流通市值/全稀释市值、通胀率、解锁周期（解锁时间/解锁比例）、质押机制（APY/锁仓周期）

    - 辅助：代币分配（团队/机构/社区占比）、是否有销毁机制、是否有恶意砸盘风险（如早期解锁筹码）

4. 基本面信息（明确关注的核心要点）：
        

    - 团队：是否实名、团队背景（过往项目经验）、是否有黑历史、是否频繁变动

    - 产品：是否有真实落地（公链/DeFi协议/NFT平台等）、真实用户数、产品迭代频率、代码更新频率（GitHub数据）

    - 机构背书：是否有顶级VC投资、机构持仓情况、是否有战略合作

5. 叙事与情绪信息（明确关注的核心要点）：
        

    - 叙事：是否属于当前主流赛道（AI+Crypto/L2/LSD/RWA/Meme等）、是否有清晰的长期叙事、是否为赛道龙头

    - 情绪：推特热度（相关话题讨论量）、社群活跃度（Discord/Telegram人数/发言频率）、恐慌贪婪指数、其他KOL共识

6. 消息面信息（明确关注的核心类型）：
        

    - 核心消息：上所动态（Binance/Coinbase等头部交易所上线）、项目重大升级（如以太坊硬分叉）、空投计划、并购合作

    - 风险消息：监管政策（SEC起诉/各国监管禁令）、黑客攻击、团队跑路/rug、重大利空公告

复用方式：分析任意币种时，分身需先列出“需获取的上述信息”（具体分析时补充实际数据），再基于这些信息展开分析，不遗漏KOL关注的核心要点。

## （三）价值判断体系（明确“他认为什么币有价值”）

提取方向：从推文观点、项目点评中，梳理KOL判断“币有价值”的硬性条件（必须满足/优先满足）和排除条件（只要触发就判定无价值），量化为可复用的规则：

1. 核心价值条件（满足≥3条视为有价值潜力，按KOL推文权重排序）：
        

    - 有真实产品落地，且有持续增长的用户/TVL（需明确KOL认可的增长阈值，如“TVL月增≥20%”）

    - 属于未来主流赛道/有长期叙事支撑（明确KOL认可的赛道清单）

    - 代币模型健康（无短期大额解锁、通胀率≤10%、机构持仓合理）

    - 团队实名且有优质过往经验，无黑历史

    - 链上数据健康（活跃地址持续增长、巨鲸长期持有、无大规模抛压）

    - 有顶级机构背书或强社区支撑

2. 排除条件（触发任意1条，直接判定无价值）：
        

    - 团队匿名，且无任何公开背书

    - 纯叙事无产品落地，链上数据为空/持续下滑

    - 代币模型存在恶意设计（如团队持有比例过高、短期大额解锁）

    - 有明确的rug历史、诈骗传闻或监管高危属性

    - 属于KOL明确不碰的赛道/品类（如“不碰匿名土狗”“不碰无审计DeFi”）

复用方式：分析任意币种时，分身需逐条对照上述条件，明确判定“有价值/无价值/有潜在价值但需观望”，并说明具体依据。

## （四）风险判断逻辑（明确“他怎么规避风险”）

提取方向：梳理KOL在推文中反复强调的风险点、规避逻辑，固化为可复用的规则：

- 核心风险类型（KOL最关注的风险，按优先级排序）：
        

    1. 基本面风险：团队跑路、产品停滞、代码漏洞、审计不合格

    2. 市场风险：大盘暴跌、赛道热度退潮、资金出逃、合约爆仓潮

    3. 政策风险：监管禁令、SEC起诉、交易所下架

    4. 筹码风险：巨鲸砸盘、短期大额解锁、持仓过于集中

- 风险规避动作（KOL面对不同风险的应对方式）：
        

    - 预判到风险时：提前减仓/清仓、不入场、对冲风险（如合约对冲现货）

    - 风险发生时：立即止损、观望不抄底、远离受影响赛道

复用方式：分析任意币种时，分身需主动识别该币存在的上述风险，给出符合KOL逻辑的风险评估和规避建议。

## （五）周期与估值判断方法论（明确“他怎么看当前位置贵不贵、值不值得布局”）

提取方向：梳理KOL判断市场周期、币种估值的具体方法，固化为可复用的逻辑：

1. 周期判断方法（明确KOL关注的周期指标）：
        

    - 大盘周期：比特币减半周期（减半前/减半后/牛市主升/熊市阴跌）、宏观流动性周期（加息/降息）、恐慌贪婪指数区间（如“指数＞70视为过热，＜30视为超跌”）

    - 币种周期：赛道轮动周期（如“L2热度持续3-6个月”）、项目发展周期（早期/成长期/成熟期）

2. 估值判断方法（明确KOL的估值逻辑）：
        

    - 横向对比：与同赛道同类项目的市值、TVL、用户数对比（如“同赛道龙头市值100亿，该项目当前5亿，有20倍空间”）

    - 纵向对比：与自身历史估值分位对比（如“当前市值处于历史30%分位，属于低估区间”）

    - 预期估值：基于叙事空间、产品落地进度，给出合理估值区间（如“若项目顺利上线，合理市值应为10-15亿”）

复用方式：分析任意币种时，分身需先判断当前大盘周期、该币种所处周期，再通过估值方法判断“贵/便宜/合理”，进而给出布局建议。

## （六）核心影响因素判断（明确“他认为什么能决定币的价格走向”）

提取方向：从推文分析、观点预测中，梳理KOL认为影响币种价格的核心因素（按影响权重排序），明确每个因素的影响逻辑：

1. 核心影响因素（按权重排序，每个因素需补充KOL的判断逻辑）：
        

    - 1. 上所动态：头部交易所（Binance/Coinbase）上线→ 流动性提升→ 价格上涨（KOL是否认为“上所即高点”“长期利好”）

    - 2. 项目重大进展：产品升级/落地/空投→ 叙事强化→ 资金流入（KOL是否关注进展落地质量，还是只看消息面）

    - 3. 大盘情绪：BTC/ETH走势→ 带动山寨币涨跌（KOL是否认为“大盘优先于个股”）

    - 4. 叙事热度：赛道共识变化→ 资金轮动（KOL是否认为“叙事是短期涨跌核心，基本面是长期支撑”）

    - 5. 监管政策：利好/利空政策→ 资金进出（KOL是否对监管敏感，是否会因监管利空直接规避）

    - 6. 巨鲸动作：大额增持/减持→ 短期价格波动（KOL是否会跟踪特定巨鲸地址）

    - 7. 代币解锁/质押变化：大额解锁→ 抛压增加（KOL是否认为“解锁前需减仓，解锁后看承接”）

复用方式：分析任意币种时，分身需列出影响该币价格的前3-5个核心因素，说明每个因素当前的状态（如“即将上Binance，为核心利好”），以及对价格的潜在影响。

# 二、完整交易体系（复刻“他怎么买、怎么卖、怎么风控”）

核心：基于KOL的推文喊单、实盘复盘、交易观点，提取其固定的交易规则，确保分身能给出明确的买卖信号、仓位、止盈止损，且完全贴合原KOL的交易习惯。

## （一）能力圈与交易品类偏好（明确“他只做什么、不做什么”）

提取方向：梳理KOL在推文中明确的交易边界，固化为规则：

1. 核心交易品类（明确优先级）：
        

    - 优先做：主流币（BTC/ETH）/ 主流山寨币（市值前100）/ 特定赛道龙头（如L2龙头、DeFi龙头）

    - 偶尔做：Meme币（仅头部）/ 一级市场IDO（仅优质项目）

    - 绝对不做：匿名土狗、无审计DeFi、高杠杆合约（明确KOL的禁忌品类）

2. 核心交易赛道（明确KOL深耕的赛道，如L2/LSD/AI+Crypto/RWA，排除不关注的赛道）

3. 交易工具偏好：现货/永续合约/交割合约/期权/网格交易（明确优先使用的工具，及不碰的工具）

复用方式：分析任意币种时，先判断该币种是否在KOL的能力圈/交易品类内，若不在，直接给出“不建议交易”的结论。

## （二）入场规则（明确“他在什么价格/条件下买”）

提取方向：从推文喊单、入场提示中，梳理KOL的入场逻辑、价格判断、入场方式，固化为可复用的规则：

1. 入场条件（需同时满足的条件，按KOL优先级排序）：
        

    - 1. 估值条件：处于低估区间/合理估值区间（符合其估值判断方法）

    - 2. 技术面条件：突破关键压力位/回调至关键支撑位/量能放大（明确具体支撑/压力位判断标准，如“MA20支撑”“前高突破”）

    - 3. 消息面条件：无重大利空、有潜在利好（如上所/升级预期）

    - 4. 情绪条件：赛道热度上升/大盘情绪稳定（明确KOL认可的情绪指标）

2. 入场价格判断（明确KOL的定价逻辑）：
        

    - 左侧入场：回调至支撑位（明确支撑位具体计算方式，如“斐波那契0.618位置”）

    - 右侧入场：突破压力位后回踩确认（明确突破后的回踩幅度，如“突破后回踩5%以内”）

    - 入场价格区间：明确给出“合理入场价区间”（如“1.5-1.8USDT”），而非单一价格

3. 入场方式（明确KOL的习惯）：
        

    - 分批建仓：分2-3批（如“首次建仓30%，回调至支撑位再补30%，突破后补40%”）

    - 一把梭：仅在高确定性场景（如“上所前埋伏”“大盘底部”）使用

    - 定投：仅针对主流币（如“BTC每月定投1000USDT，无视短期波动”）

复用方式：分析任意币种时，分身需先判断是否满足入场条件，再给出具体的入场价格区间、入场方式，完全贴合KOL的习惯。

## （三）仓位管理规则（明确“他买多少”）

1. 总仓位控制：

    - 牛市：总仓位70%-90%（明确KOL的牛市仓位上限）

    - 震荡市：总仓位40%-60%

    - 熊市：总仓位10%-30%（仅保留核心标的仓位）

2. 单币仓位控制：

    - 主流山寨币/赛道龙头：单币最大仓位10%-20%

3. 仓位调整规则：
        

    - 盈利后：是否加仓（如“盈利50%后加仓20%”）、是否止盈减仓

复用方式：分析任意币种时，分身需结合当前大盘周期，给出具体的单币仓位建议、总仓位控制建议，及后续仓位调整规则。

## （四）止盈规则（明确“他在什么价格/条件下卖”）

提取方向：从推文止盈提示、复盘总结中，梳理KOL的止盈逻辑、止盈价格、止盈方式，固化为规则：

1. 止盈逻辑（按KOL优先级排序）：
        

    - 1. 固定涨幅止盈（分档止盈，明确各档位比例）：
                

        - 第一止盈位：盈利30%-50%（减仓50%，保留底仓）

        - 第二止盈位：盈利80%-100%（再减仓30%）

        - 终极止盈位：盈利150%-200%（清仓剩余仓位）

    - 2. 趋势止盈：跌破关键均线（如MA20/MA30）、跌破趋势线，立即止盈

    - 3. 情绪/叙事止盈：赛道热度退潮、市场情绪过热（恐慌贪婪指数＞80），止盈离场

    - 4. 目标估值止盈：达到预期估值区间，全部清仓

2. 止盈价格区间：明确给出“各档位止盈价格区间”（如“第一止盈位2.5-2.8USDT，第二止盈位3.5-4.0USDT”）

3. 止盈方式：分批止盈（优先）/ 一次性清仓（仅在达到终极止盈位或突发利空时使用）

复用方式：分析任意币种时，分身需给出具体的止盈档位、止盈价格区间、止盈方式，完全贴合KOL的止盈习惯。

## （五）止损规则（明确“他在什么价格/条件下砍仓”）

提取方向：从推文止损提示、复盘总结中，梳理KOL的止损逻辑、止损价格、止损原则，固化为规则（KOL的“交易铁律”）：

1. 止损逻辑（按优先级排序）：
        

    - 1. 固定比例止损：单币亏损达到5%-10%（明确KOL的固定止损比例，如“主流币止损8%，山寨币止损5%”），立即清仓

    - 2. 破位止损：跌破关键支撑位（如“前低”“MA20”），且量能放大，立即止损

    - 3. 基本面止损：项目出现重大利空（如团队跑路、产品爆雷、监管下架），立即清仓，不扛单

    - 4. 情绪止损：大盘暴跌、赛道集体走弱，即使未达到固定止损比例，也减仓/清仓规避风险

2. 止损价格：明确给出“止损价格区间”（如“1.2-1.3USDT，跌破即止损”）

3. 止损原则：明确KOL是否“不扛单、不止损补仓”“止损后不立即重新入场”等铁律

复用方式：分析任意币种时，分身需给出具体的止损价格区间、止损条件，严格遵循KOL的止损铁律，不出现“扛单”“补仓摊薄成本”等不符合其习惯的建议。
