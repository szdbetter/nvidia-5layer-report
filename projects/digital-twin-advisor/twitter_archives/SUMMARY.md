## 数据获取方法说明

本次使用 `bird search` 工具，按月分片抓取绕过Twitter的深度限制：

```bash
# 按月分片策略
for month in 2024-07 2024-08 ... 2025-06; do
  bird search "from:USER since:$month-01 until:$month-01+1month"
done
```

相比直接 `user-tweets`，这种方法能绕过40-50条的深度限制，抓取到近一年的完整数据。
