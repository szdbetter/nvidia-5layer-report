## 核心方法论

### 1. 多付费最后少付费
> 我的理念是：多付费，最后反而是少付费。
- 每月花$1000在AI工具上，但换来的是一个<10人团队做超大产出
- 用对工具比省钱更重要，用错模型的时间成本远超工具费用

### 2. AI工具分工论（核心框架）
| 工具 | 用途 | 不用于 |
|------|------|--------|
| Claude Code (Opus/Max) | 产品开发、代码、架构决策 | - |
| ChatGPT | 头脑风暴、发散思维、聊天 | 做执行决策（马屁太多不可信） |
| NotebookLM | 精准知识库、文献整理、报告 | - |
| Gemini | 跟新功能、教育优惠、长上下文 | - |
| Grok | 抓取Twitter/X实时信息 | - |
| Comet浏览器 | 信息读取、竞品监控 | 内容生成 |
| AI Studio | 快速Demo验证、Prompt调试 | 正式产品 |
| Lovart | 跨境电商产品图、图层精修 | - |
| YouMind | 手绘风格信息图、YouTube转笔记 | - |
| n8n | 自动化工作流搭建 | - |

### 3. 三层AI成本公式
> 想要生成好的结果，成本 = 大模型（花钱）+ Token（花钱）+ 精准数据/Skills/MCP（花钱）
- 省模型成本 = 省错了地方
- 省数据接入成本 = AI会乱说话（幻觉）

### 4. Claude稳定使用方案
- **官方Max方案**：Claude Code Max，贵但稳，$129-200/月
- **Google AI Ultra方案**：通过Google Antigravity使用Claude，B2B正规渠道，抗封号
- **坚决不做**：通过第三方工具把Claude能力抽出来变API，大概率被封

### 5. 养"小龙虾"理念（专属AI顾问）
- 不要逮住一只AI啥都问，垂直才能专业
- 为不同业务场景养专属Agent：市场分析师、跨境专家、股票专家、自媒体专家、AI教练
- 给每只Agent接入对应数据源（Reddit/Softime/MCP），避免幻觉

### 6. AI时代竞争力判断
- AI平掉了经验门槛，无法平掉判断力
- 未来只有两种人：会判断的人 vs 被判断的人
- 工作流搭得再好，业务理解和判断力才是护城河

## 工具使用细节（实操经验）

### Claude/Antigravity
- Claude Code + n8n MCP = 工作流做起来跟喝水一样
- Antigravity登录卡住：开TUN模式/虚拟网卡模式，处理TCP层流量
- 绝对不要走Antigravity Tools抽API路线，已被封验证

### NotebookLM
- 给的数据是精准的，回答就精准；给一坨屎，输出就是屎山
- 按类别建知识库，不要混用（读书库/亚马逊库/行业库分开）

### n8n
- 部署在官网（Cloud版），$24欧/月
- 配合Claude Code + MCP，报错自动修复

### Lovart
- 用Nao Banana 2生成产品图 → 图层编辑二次精修 → 直接上Listing
- 解决了AI生成图"最后一步精修"的痛点

## 金句精选
1. 多付费，最后反而是少付费。
2. AI把经验门槛平掉了，但无法平掉你的判断力。
3. 用对模型，和随便抓一个来用，结果可以天差地别。
4. Demo用AI Studio，产品用Claude Code。
5. 垂直才能专业，不要逮住一只小龙虾啥都问。
6. 能不能超过AI的成本上限，是每个打工人要认真面对的问题。
7. 你没有见识过别人的用法，还在和大模型直接对话聊天的初级阶段，那自然无法感受到AI真正的强大。
8. 不要觉得自己很勤奋，你现在的勤奋在AI面前不值得一提。
