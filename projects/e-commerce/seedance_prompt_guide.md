# Seedance 图生视频 Prompt 指南
> 来源：火山方舟官方文档 https://www.volcengine.com/docs/82379/1587797
> 整理时间：2026-03-15，适用模型：doubao-seedance-1-0-lite-i2v-250428

---

## 一、图生视频 Prompt 核心公式

```
Prompt = 主体 + 运动，背景 + 运动，镜头 + 运动
```

**5条基础原则：**
1. **减少静止描述**：图片本身已提供场景，只描述「运动的部分」，不要重复描述已有静止画面
2. **简单直接**：用简单词语和句子，模型会自动扩写
3. **突出特征 + 明确程度副词**：「快速」「大幅度」「剧烈」必须写明，否则模型自行判断
4. **遵从图片内容**：不要与图片事实矛盾（图中是男人别写女人）
5. **负向提示词无效**：模型不响应「不要...」类指令

## 二、真人感提升技巧（解决AI感太重的问题）

### 降低AI感的关键操作：

**① 使用手持镜头**
```
手持镜头，画面微微抖动体现手持感，跟随...
```

**② 避免"完美"描述，加入真实感细节**
- ❌ 不写：「完美的步伐」「精准的动作」
- ✅ 改写：「步伐自然」「微微晃动」「随意走动」

**③ 明确眼神自然**
```
她眼神自然，...（加上"眼神自然"能避免死鱼眼）
```

**④ 场景要「生活化」而非「广告感」**
- ❌ 广告感：完美光线、对称构图
- ✅ 真实感：室内冷白顶光、轻微曝光不足、自然散乱背景

**⑤ 运镜选「不固定镜头」+ 手持/跟随**
```
基础参数：不固定镜头（camera_fixed: false）
Prompt：手持镜头跟随，画面微微抖动
```

## 四、运镜词汇表

| 效果 | Prompt关键词 |
|------|-------------|
| 手持真实感 | 手持镜头，画面微微抖动 |
| 跟随拍摄 | 镜头跟随，紧随主体移动 |
| 镜头切换 | 镜头切换（明确写出来） |
| 拉近 | 镜头推近，特写 |
| 拉远 | 镜头拉远，展示全景 |
| 下摇 | 镜头下摇，从脸部到全身 |
| 环绕 | 镜头环绕，360度展示 |
| 平移 | 镜头缓慢向左/右移动 |

**注意：有运镜提示词时，必须选「不固定镜头」（camera_fixed: false）**

## 五、程度副词使用（让动作更明显）

```
快速 / 剧烈 / 大幅度 / 高频率 / 强力 / 疯狂
```

- 「模特走路」→ 「模特轻快地大步走路」
- 「头发飘动」→ 「头发随风大幅度飘动」
- 建议：适当夸大程度，效果更接近预期

## 六、常见问题修复

### 问题1：AI感太重 / 像广告片
**根因**：固定镜头 + 完美光线 + 标准构图
**解决**：
- 改用「手持镜头，画面微微抖动」
- 加「眼神自然」「表情放松」
- 光线描述改为「冷白顶光，自然偏暗」

### 问题2：模特眼神奇怪（死鱼眼）
**解决**：在Prompt中明确写「她眼神自然」或「她眼睛看向前方，神情自然」

### 问题3：动作幅度不够
**解决**：加程度副词，如「快速」「大幅度」「有力地」

### 问题4：重要动作没有执行
**原因**：Prompt中静态描述太多，稀释了运动指令
**解决**：删除图片中已有的静止信息，只保留运动描述

### 问题5：肢体崩坏（手脚变形）
**解决**：Prompt中避免描述手部/脚部细节，或多抽卡选优

## 八、API调用参数速查

```python
payload = {
    "model": "doubao-seedance-1-0-lite-i2v-250428",  # 图生视频
    "content": [
        {"type": "text", "text": "你的prompt"},
        {"type": "image_url", "image_url": {"url": "图片URL或base64"}, "role": "reference_image"}
    ],
    "ratio": "9:16",        # 比例
    "duration": 10,          # 时长（秒）
    "camera_fixed": False,   # 不固定镜头（有运镜时必须False）
    "watermark": False,      # 去水印
    # "generate_audio": True  # 生成音频（lite版不支持）
}
```

**轮询查询：**
```
GET https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks/{task_id}
每20s轮询，status=succeeded时取 content.video_url（24h有效）
```
