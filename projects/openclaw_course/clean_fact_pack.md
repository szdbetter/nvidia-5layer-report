## 8个最重要主题

### 1. OpenClaw 是什么
- 开源、自托管的 AI Agent 系统，MIT 协议
- 核心定位：让 AI 从「聊天工具」变成「能自主执行任务的数字员工」
- 吉祥物是龙虾，中文社区称使用 OpenClaw 为「养虾」
- 与 ChatGPT 的核心区别：ChatGPT 是「顾问」（你问它答），OpenClaw 是「员工」（主动执行任务）

### 2. 增长数据与热度
- GitHub Stars 28万+（截至2026年3月9日），已超越 React 成为全球第一软件项目
- 从0到28万 Stars 只用了不到5个月，React 用了超过10年
- 单日最高增长 9000 Stars（平均每10秒一个开发者点 Star）
- 贡献者 1075+，社区 Skills 13700+

### 3. 三层架构设计
- Gateway（中央控制平面）：维护 WebSocket 服务、管理 Session、调度 Agent，默认绑定 ws://127.0.0.1:18789
- Node（设备端执行层）：负责本地操作（摄像头、录屏、系统命令等）
- Channel（消息渠道接入层）：连接 20+ 即时通讯平台（WhatsApp、Telegram、Discord、飞书、钉钉等）
- 默认 Loopback-First 设计：只绑定 localhost，不开放外网端口，天然安全

### 4. 四层记忆系统
- SOUL（不可变内核）：Agent 的人格、价值观，存储在 SOUL.md，创建后不应修改
- TOOLS（动态工具）：当前可用的工具和技能列表，随安装动态变化
- USER（语义长期记忆）：用户偏好、决策、历史事实，存储在 MEMORY.md，支持语义搜索
- Session（实时情景）：当前对话的实时上下文，Token 耗尽时被压缩
- 自动记忆保存机制：当 Session 接近 Token 上限时，自动触发 Pre-Compaction 将重要记忆写入文件

### 5. 部署方式选择
- 本地安装（npm）：适合开发者，两条命令搞定
- Docker 部署：适合需要环境隔离、方便迁移的场景
- 国内云厂商一键部署：阿里云（9.9元/月）、腾讯云（约17元/月）、百度云（0.01元首月）、火山引擎（9.9元/月）等
- 扣子编程：无需服务器，49元/月起，零门槛

### 6. 模型配置与成本
- 支持十余家模型提供商：Anthropic Claude、OpenAI GPT、Google Gemini、DeepSeek、智谱 GLM、通义千问、豆包、Kimi 等
- 国产模型性价比突出：DeepSeek-V3.2 输入价格仅 $0.14/百万 tokens，是 Claude Sonnet 的 1/20
- 核心省钱策略：配置 Fallback 链（如 Sonnet → Haiku → DeepSeek），可降低 80-95% API 成本
- 必须设置预算上限：OpenClaw 的多轮工具调用可能消耗传统聊天几十到上百倍 Token

### 7. Skills 系统与安全风险
- Skills 是能力扩展单元，三层优先级：项目级 > 用户级 > 内置
- ClawHub 市场有 13729 个 Skills，但超过 50% 被判定为垃圾/重复/低质量，约 20% 被标记为恶意
- ClawHavoc 供应链攻击：2026年1-2月，135000+ 设备受影响，800+ 恶意 Skills 被植入信息窃取木马
- 安全建议：安装前审查源码、使用 SecureClaw 扫描、优先使用 awesome-openclaw-skills 精选列表

### 8. 已知安全事件
- CVE-2026-25253（CVSS 8.8/10）：WebSocket origin 绕过漏洞，可导致远程代码执行，13.5万暴露实例中5万+可被直接攻击
- 谷歌封号风波：2026年2月大规模封禁 OpenClaw 用户 Google 账号
- 3万+台未认证暴露实例：Gateway 端口对任何人开放，API 额度和个人数据面临风险
- 工信部安全预警：2026年3月8-9日，工信部和 CNCERT 正式发布 OpenClaw 安全风险预警

## 小白最该关注的 10 个信息点

1. **OpenClaw 是「员工」不是「顾问」**：它会主动执行任务，不是简单问答
2. **部署门槛比想象中低**：阿里云/腾讯云一键部署，3步完成，9.9元/月起
3. **国产模型完全可用**：DeepSeek-V3.2 和 GLM-5 是性价比最高的选择
4. **必须设置消费限额**：不做控制可能一觉醒来收到 1100 美元账单
5. **Fallback 链是省钱核心**：配置三级 Fallback 可降低 80-95% 成本
6. **Skills 安装要谨慎**：ClawHub 上 20% Skills 是恶意的，安装前必须审查源码
7. **默认绑定 localhost 是安全设计**：不要直接暴露 Gateway 到外网
8. **v2026.3.7 起强制要求 Gateway 认证**：不配置无法启动
9. **微信接入有门槛**：个人微信没有官方 API，需通过企业微信中转
10. **Claude Code 是互补关系**：OpenClaw 管生活自动化，Claude Code 管代码，两者可以桥接使用
