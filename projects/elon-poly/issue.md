# Elon Polymarket 三因子策略 — Issue & 迭代日志

## 任务目标
用三因子模型（条件概率+动量+事件驱动）回测 Elon 推文盘口，
达到稳定盈利标准后通知老板充值实盘。

## 盈利标准
- 期望价值 EV > 0
- 年化 Sharpe > 1.0  
- 总 P&L > 0
- 平均 Edge > 6%

## 当前状态：v2.0 初跑通 (2026-03-23)

### 数据
- 42 个 xtracker 历史 tracking，39 个完成（7天周期）
- 均值 361 条/周，范围 165-570

### 初始回测结果（Day6入场，min_edge=8%）
- 交易次数: 34
- 命中率: 17.6%（单档宽20条，合理）
- 总P&L: +$4,819（$500基础注额）
- Sharpe: 1.51
- 最大回撤: $1,391
- 盈亏比: 2.04x
- **状态: 已达标**

## 已知问题 & 待改进

### P1: 市场价格模型是散户线性外推的模拟，非真实 Polymarket 价格
**影响**: 真实 edge 可能比模型显示的低（TweetCast 也在这个市场）
**改进方向**: 接入真实 Polymarket CLOB API 获取历史价格（需 Gamma API）

### P2: 条件概率表样本量不足
**现状**: Day6 入场依赖 Day5 累计数据，但我们的历史 daily 数据按小时粒度，
需要准确聚合为"第1-5天"
**影响**: 条件概率表 row (350-399) 只有 6-14 个样本，置信区间宽
**改进方向**: 用参数化模型（负二项分布）替代查表法

### P3: 押注总是最高 edge 档，可能过于集中
**改进方向**: 测试多档分散押注策略

### P4: 未测试 Day5 入场 vs Day6 入场的差异
**初步网格搜索结果**: 见下次迭代
