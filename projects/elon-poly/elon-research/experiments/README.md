# 首轮实验约束

本目录仅用于记录首轮事件研究与严格回放实验，不承载生产交易逻辑。

## 范围

- 只研究 Elon 推文事件与 Polymarket 市场反应的关系。
- 只允许使用已采集并落盘的历史快照，不接入实时行情或实时推文流。
- 首轮实验目标是验证是否存在可重复的研究 edge，不输出自动下单能力。

## 数据约束

- 原始数据必须先通过 `ingest` 命令写入 `datasets/raw/`。
- 标准化阶段只能读取原始快照，不允许跨阶段直接请求外部接口。
- 推文时间戳与市场时间戳都必须保留原始来源字段，便于回放审计。

## 回测约束

- 只允许事件后回放，不允许使用未来信息。
- 所有信号必须基于事件发生时可见的数据构建。
- 首轮只接受最小可复现实验，策略参数必须显式记录，不能手工挑选样本。

## 输出要求

- 实验结论、假设、失败原因统一记录到 `datasets/reports/`。
- 所有新增文档、注释、报告结论统一使用中文。
