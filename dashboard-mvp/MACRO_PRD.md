# 🌊 霍尔木兹宏观引擎监控页 — 产品需求文档 v1.0

## 页面入口
- URL: `/macro`
- 从主 Dashboard 左侧导航 📊 Polymarket > 🌊 宏观引擎 进入
- 独立全屏页面，有返回主 Dashboard 的按钮

## 设计原则
- **与主页风格 100% 统一**：白色内容区 `#f0f2f5`，白色卡片，`#001529` 顶栏
- **无黑色/深色背景**（整页白色系，参考 Ant Design Pro 的分析页）
- **数据即界面**：每个区域要么有数据，要么有明确的加载/空状态

## 页面结构（从上到下）

### 区域 A：顶栏（与主页完全一致）
```
┌─────────────────────────────────────────────────────┐
│ 🌊 霍尔木兹宏观引擎          ← 返回  🕐 北京时间   │
└─────────────────────────────────────────────────────┘
```
- 背景 `#001529`，白色文字
- 右侧：实时北京时间 + 30秒倒计时刷新

### 区域 B：信号状态卡（满宽白色卡片）
```
┌─────────────────────────────────────────────────────┐
│  🚨 冲突升级 · 坚持持有NO仓，低位加仓               │
│  最后更新：2026-03-15 22:00 北京时间  |  模拟数据    │
└─────────────────────────────────────────────────────┘
```
- 信号文字颜色：冲突=红 `#ff4d4f`，缓解=绿 `#52c41a`，静默=橙 `#faad14`，无数据=灰
- 白色卡片，左侧竖线颜色跟信号一致
- Mock 模式显示橙色 badge「⚠ 模拟数据」

### 区域 C：3 列指标卡
```
┌──────────┐ ┌──────────┐ ┌──────────┐
│ 🚢 通行量 │ │ ⚓ 等待   │ │ 🛡️ 战争险│
│  ↓13.9%  │ │ 69.4小时  │ │ -14.1%   │
│ (当前86%) │ │          │ │ (下降)    │
└──────────┘ └──────────┘ └──────────┘
```
- 白色卡片，数字大号 28px
- 通行量下降=红色↓，上升=绿色↑
- 战争险上升=红色↑，下降=绿色↓

### 区域 D：2 列图表
```
┌─────────────────┐ ┌─────────────────┐
│ 📊 通行量 7日    │ │ ⚠️ 战争险走势    │
│  折线+柱状图     │ │  面积图          │
│                 │ │                 │
└─────────────────┘ └─────────────────┘
```
- 白色卡片包裹 ECharts 图表
- 数据从 localStorage 累积
- 空数据时显示「📊 暂无历史数据，等待引擎运行后自动累积」

### 区域 E：Polymarket 盘口扫描
```
┌─────────────────────────────────────────────────────┐
│ 🎯 Polymarket 美伊停战 · 盘口扫描                    │
│  3月31日停战 YES: 14.5%  ✅ 盘口合理                  │
│  4月30日停战 YES: 35.5%  ⚠️ 可能高估                 │
└─────────────────────────────────────────────────────┘
```
- 白色卡片
- 从 Polymarket API 实时拉取
- 加载失败显示「盘口数据加载中...」

### 区域 F：引擎详情
```
┌─────────────────────────────────────────────────────┐
│ 📋 引擎详情                                         │
│ 告警内容：[宏观引擎] 🚨 霍尔木兹冲突升级...          │
│ 运行模式：模拟 | Fiona节点：异常                     │
└─────────────────────────────────────────────────────┘
```
- 白色卡片，告警文本用浅红背景 `#fff2f0` 边框 `#ffccc7`

## 加载状态
- 所有卡片初始显示脉冲骨架屏
- 数据加载完成后替换为真实内容
- 失败时显示红色提示 + 重试按钮

## 技术约束
- 纯 HTML + CSS + JS（不引入 React）
- ECharts 5.5（CDN）
- Bootstrap 不使用（全部自定义 CSS 与主页统一）
- 30 秒自动刷新
