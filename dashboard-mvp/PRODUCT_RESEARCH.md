## 二、产品调研 — 三个核心问题

### Q1: 这套系统最终要达到什么目标？

**我的理解：**
你需要一个 **Polymarket 美伊停战策略决策辅助系统**，核心功能：
1. 实时监控霍尔木兹海峡油轮通行量 + 劳合社战争险费率
2. 基于硬指标自动判断「冲突升级/封锁缓解/信号静默」
3. 与 Polymarket 盘口对比，识别错误定价
4. 告警推送到 Discord，支持手机端查看
5. Dashboard 可视化整体局势

| 数据 | 来源选项 | 状态 |
|------|----------|------|
| 油轮通行量 | JMIC API / MarineTraffic / Kpler | ❌ 未接入 |
| 战争险费率 | 劳合社 API / 保险经纪商数据 | ❌ 未接入 |
| Polymarket 盘口 | gamma-api.polymarket.com（免费） | ✅ 可用 |

### Q3: 交付标准 — 什么才算「完全可跑」？

我理解的验收标准：
1. ✅ Dashboard 主页（ClawLabs Home）— 项目概览 + 任务看板 + 系统状态
