### 地缘危机监控系统学习（@xingpt 公开案例）
- 核心方法论可复用至 Polymarket 美伊标的决策优化：
  1. **信源直达策略**：跳过二手媒体，直接抓取 JMIC、Windward 等一手海事/官方信源，引入波斯语本土信源做交叉验证，获取信息差。
  2. **硬指标降噪**：盯住核心可量化指标（如海峡每日通行量/停火事件），过滤杂散新闻噪音，锚定决策锚点。
  3. **Agent 架构隔离**：投资决策链路与主会话物理隔离，避免上下文污染。
  4. **分级数据流**：优先走 API 拉取宏观/交易数据，强制限制网页搜索次数，避免 context 膨胀死循环。
