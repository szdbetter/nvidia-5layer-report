# Discord 投研发帖模板

## 使用原则
- 普通正文使用 Markdown + Emoji 语义着色
- 若需要彩色视觉效果，使用 Embed 卡片，不依赖彩色字体
- 研究场景分为：美股 / 港股；发帖时标题和字段随场景切换

---

## 模板 A：Markdown 速发版（通用）

```md
# [市场] [赛道] 一阶段筛选结论

**一句话结论**：本轮 Alpha 不在最热情绪票，在 [卖铲人/基础设施/绑定式生态]。

## 最新排序
- 🥇 **[标的1 / 代码]**： [一句话逻辑]
- 🥈 **[标的2 / 代码]**： [一句话逻辑]
- 🥉 **[标的3 / 代码]**： [一句话逻辑]

## 为什么不是最热票
- 🎭 **[情绪龙头1]**： [过热/透支/赔率差]
- 🎭 **[情绪龙头2]**： [过热/透支/赔率差]

## 核心表
| 标的 | 定位 | 优势 | 风险 | 评级 |
|------|------|------|------|------|
| [代码1] | [直接/映射/情绪] | [一句话] | [一句话] | 🟢/🟡/🟠/🔴 |
| [代码2] | [直接/映射/情绪] | [一句话] | [一句话] | 🟢/🟡/🟠/🔴 |

## 下一步
- 二阶段深挖：**[代码1] / [代码2]**
- 检查项：估值分位 / 营收质量 / 筹码 / 催化剂 / 熔断条件
```

## 模板 B：港股版 Markdown

```md
# 港股 AI Agent / 机器人 一阶段结论

**一句话结论**：这轮港股 Alpha 优先看 **[商汤/金山云/其他]**，不是盲追 **[MiniMax/智谱/其他情绪票]**。

## 排名

- 🥉 **[代码3]**：主题弹性大，但兑现慢

## 分类
- ✅ **直接受益**：[列表]
- 🔄 **映射受益**：[列表]
- 🎭 **情绪龙头**：[列表]

## 港股特有风险
- 港股通：[可买/不可买/待核实]

- 二阶段建议：**[代码1] / [代码2]**
- 回避名单：**[代码X]**（原因：[一句话]）
```

## 模板 C：Embed 字段版（适合机器人/消息工具）

### Embed 结构建议
- **标题**：`[市场] [赛道] Alpha 结论`
- **颜色**：
  - 绿色：强跟踪
  - 黄色：重点关注
  - 橙色：观察等待
  - 红色：回避
- **字段**：
  1. 一句话结论
  2. Top 3 排名
  3. 情绪票排除理由
  4. 二阶段名单
  5. 风险提示

### 字段文案示例
- **一句话结论**：Alpha 在基础设施，不在最热新股。
- **Top 1**：`[代码]` — [一句话逻辑]
- **Top 2**：`[代码]` — [一句话逻辑]
- **Top 3**：`[代码]` — [一句话逻辑]
- **排除**：`[票名]` — 估值透支 / 筹码拥挤 / 只是情绪锚
- **二阶段**：`[代码1], [代码2]`
- **风险**：减持 / 配售 / 流动性 / 做空 / 监管

## 模板 D：长帖版（带二阶段衔接）

```md
# [市场] [赛道] Alpha Brief

## 结论先行
本轮最值得继续深挖的是 **[代码1]** 和 **[代码2]**。
原因：它们分别卡住了 **[算力/生态/供应链/具身智能]** 的真实瓶颈，而不是单纯靠情绪抬估值。

## 一阶段筛选结果
### 🥇 [代码1]
- 定位：[一句话]
- Alpha 来源：[一句话]
- 核心风险：[一句话]

### 🥈 [代码2]
- 定位：[一句话]
- Alpha 来源：[一句话]
- 核心风险：[一句话]

### 🥉 [代码3]
- 定位：[一句话]
- Alpha 来源：[一句话]
- 核心风险：[一句话]

## 为什么排除热门票
- [票名1]：热度太满，赔率下降
- [票名2]：市场已充分定价

## 二阶段检查清单
- 估值历史分位
- 收入增长质量
- 现金流与跑道
- 股东/减持/筹码
- 催化剂与熔断条件
```

## 备注
- Discord 正文不支持真正彩色字体
- 想要“彩色效果”，优先用 Embed，不要依赖 ANSI 代码块
- 做频道统一模板时，建议：**正文 Markdown，结论卡片 Embed，长文外链网页**
