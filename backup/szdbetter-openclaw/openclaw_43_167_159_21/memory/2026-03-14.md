
## 2026-03-14: Fiona Node Upgrade Troubleshooting
* **Issue**: fiona-mbp2015 node was stuck on OpenClaw 2026.3.8 and timing out on Discord RPC requests.
* **Root Cause**: An `npm install openclaw@latest` triggered via RPC was running slowly in the background, causing the node to timeout. Lingering 2026.3.8 gateway and node processes remained active.
* **Resolution**: Connected to the node directly via Tailscale SSH (`ssh ai@100.101.75.22`), used `pkill -f openclaw-` to forcefully clear the old processes, and executed `/usr/local/bin/openclaw gateway restart` to bring up the newly installed 2026.3.12 version. Note: The launch agent restart initially reported a timeout waiting for port 1979, requiring careful process monitoring.
